# realistic_user_dataset

本目录模拟 林澈（信息管理方向研二学生）在 2026-04-07 至 2026-05-30 使用 RAGNotebook 沉淀资料、会议记录、论文阅读、项目规划、课程学习和汇报材料的过程。

数据集围绕同一条研究线索展开：PKM-2026 个人知识库复用研究。文件之间共享人物、时间线、项目代号、导师反馈和阶段成果，适合直接上传到知识库体验跨来源检索、来源核对和资料复用。

## 文件类型覆盖

- `.txt`：碎片记录、访谈摘录、待办复盘、阅读摘记。
- `.md`：周计划、论文卡片、项目日志、导师同步问题、工具使用复盘。
- `.markdown`：课程笔记、研究主题索引、资料整理说明。
- `.docx`：开题讨论纪要、阶段性研究简报、导师反馈整理、项目需求说明。
- `.pdf`：讲义整理、调研报告、实验记录汇总、用户研究报告。
- `.pptx`：组会汇报、阶段成果展示、研究路线说明、知识库方法分享。

旧版 `.doc` 和 `.ppt` 未包含，因为当前解析器会提示另存为 `.docx` 或 `.pptx` 后再导入。这里仅保留当前可成功解析和切片的格式。

## 建议使用方式

1. 将全部 24 个内容文件上传到同一个知识库文件夹。
2. 以“资料复用”“模糊回忆”“导师反馈”“证据链”“组会汇报”为关键词进行检索。
3. 对比 TXT、Markdown、DOCX、PDF、PPTX 在检索结果中的来源线索和可引用程度。
